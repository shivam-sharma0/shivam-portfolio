My Last Projects :


Vaccine Management System


The Vaccine Management System is a comprehensive solution designed to streamline the process of scheduling and managing vaccine appointments. It features seamless email integration, sending automated notifications to users when appointments are booked. The system includes essential classes such as Doctor, User, Appointment, Dose, and Vaccine Center to facilitate smooth operations. The Doctor class enables healthcare professionals to manage their availability, while the User class allows individuals to book appointments and track their vaccination progress. The Appointment and Dose classes handle appointment scheduling and dose administration,respectively.Additionally, the Vaccine Center class manages the inventory and distribution of vaccines. With this project, the aim is to provide an efficient and user-friendly platform for vaccine management, promoting public health and facilitating the vaccination process.


Book My Show (Backend)


BookMyShow is a web-based ticket booking platform developed using Spring Boot, Maven, Lombok, and RESTful APIs. The project enables users to book tickets for movies, view available seats, and make secure online payments. It provides a seamless experience for users to select their preferred movie, theater, and showtime, and choose seats within the theater's seating arrangement. The system also includes an admin module that allows theater owners to register their theaters, define the structure of the theater (number of screens, seating arrangements, etc.), and manage movie showtimes. With its user-friendly interface and robust backend architecture, BookMyShow simplifies the ticket booking process and enhances the overall movie-going experience for users.


Web Search Engine


The web search engine project I developed is a powerful tool that enables users to perform searches and retrieve relevant information. It features a frontend built with HTML and CSS, providing a user-friendly interface for search queries. In the backend, Java Swing and various APIs are utilized to extract data from the JavaTpoint website. The project incorporates a DFS algorithm to navigate through web pages by following hyperlinks and extracting valuable data. This extracted data is then stored in a MySQL database, creating a structured repository. When users input search keywords, the engine retrieves related data from the database, delivering accurate and targeted search results. This project showcases the integration of frontend technologies, backend algorithms, and database management to create an efficient and comprehensive web search engine.


Snake Game


The Snake Game is an engaging sub-genre of action video games where players control a growing line, resembling a snake, and navigate it through obstacles without colliding with itself or other barriers. In this project, I have implemented the game using Java and the Swing framework, allowing for smooth graphics and user interaction. Additionally, I have utilized Jsoup for web scraping functionality and Maven for efficient dependency management. The game features the ability to start and end the game, with a game over pop-up displayed once the game concludes. With its simple yet addictive gameplay and the use of various technologies, the Snake Game offers an enjoyable and immersive gaming experience.
